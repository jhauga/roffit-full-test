Name

curl_url_cleanup - free the URL handle

Synopsis

#include <curl/curl.h>
 
void curl_url_cleanup(CURLU *handle);

Description

Frees all the resources associated with the given CURLU handle!

Passing in a NULL pointer in handle will make this function return immediately with no action.

Example

  CURLU *url = curl_url();
  curl_url_set(url, CURLUPART_URL, "https://example.com", 0);
  curl_url_cleanup(url);

Availability

Added in 7.62.0

Return value

none
