Name

libssh2_sftp_shutdown - shut down an SFTP session

Synopsis

#include <libssh2.h>
#include <libssh2_sftp.h>
 
int
libssh2_sftp_shutdown(LIBSSH2_SFTP *sftp);

Description

sftp - SFTP instance as returned by libssh2_sftp_init(3)

Destroys a previously initialized SFTP session and frees all resources associated with it.

Return value

Return 0 on success or negative on failure. It returns LIBSSH2_ERROR_EAGAIN when it would otherwise block. While LIBSSH2_ERROR_EAGAIN is a negative number, it is not really a failure per se.

See also

libssh2_sftp_init(3)
