Name

libssh2_agent_list_identities - request an ssh-agent to list of public keys.

Synopsis

#include <libssh2.h>
 
int
libssh2_agent_list_identities(LIBSSH2_AGENT *agent);

Description

Request an ssh-agent to list of public keys, and stores them in the internal collection of the handle. Call libssh2_agent_get_identity(3) to get a public key off the collection.

Return value

Returns 0 if succeeded, or a negative value for error.

Availability

Added in libssh2 1.2

See also

libssh2_agent_connect(3) libssh2_agent_get_identity(3)
